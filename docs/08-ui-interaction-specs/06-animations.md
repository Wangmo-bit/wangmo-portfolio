# 动效与赛博氛围交互稿

## 核心动效

### 1. 左栏背景联动（最重要）

- **触发**：右栏滚动，当新的作品卡片进入视口中央
- **效果**：左栏底部的径向渐变色从当前主题色过渡到新主题色
- **时长**：0.6s ease-in-out
- **实现**：CSS transition on background / gradient

```
默认（无聚焦）：纯深色背景

聚焦赛博生活：
+------------------------+
|                        |
|       (深色)            |
|                        |
|     ~~~青色微光~~~      |
+------------------------+

聚焦爪印档案：
+------------------------+
|                        |
|       (深色)            |
|                        |
|     ~~~暖橙微光~~~      |
+------------------------+

聚焦 Lorebook：
+------------------------+
|                        |
|       (深色)            |
|                        |
|     ~~~紫色微光~~~      |
+------------------------+
```

### 2. 头像边框呼吸光

- 头像外圈有一道细光环
- 光环颜色跟随当前主题色
- 缓慢呼吸明暗变化（3s 周期）
- CSS animation: box-shadow pulse

### 3. 卡片 Hover 效果

- translateY: -4px（上浮）
- 边框亮度增强（opacity 0.3 → 0.8）
- box-shadow 扩散微光
- transition: 0.2s ease

### 4. 页面加载动画

- 左栏内容依次淡入（staggered fade-in）
  - 头像 → 姓名 → 简介 → 社交链接，间隔 0.1s
- 右栏卡片依次从右侧滑入
  - 间隔 0.15s，translateX 20px → 0

## 克制原则

- 不使用 3D 效果、粒子系统、Canvas 动画
- 不使用 Matrix 数字雨、故障闪烁等重特效
- 赛博感通过 **配色 + 微光 + 渐变** 传达，而非花哨动画
- 所有动效 `prefers-reduced-motion` 时禁用

## 不做（v1.0）

- 鼠标跟随光效
- 打字机文字效果
- 背景粒子 / 星空
- 页面切换过渡动画
