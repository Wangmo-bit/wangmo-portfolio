# 语言切换交互稿

## 切换控件

### 桌面端

- 位置：左栏底部
- 样式：文字按钮 `中文 / EN`
- 当前语言高亮（主题色），另一个低对比度
- 点击切换，无页面刷新（前端切换）

### 移动端

- 位置：头部区域右上角
- 样式：紧凑文字按钮 `中 / EN`

## 切换行为

- 切换时所有文案同时更新（姓名、简介、作品描述、状态标签、报道标题）
- 过渡效果：文字 fade 切换（opacity 0 → 1，约 0.2s）
- URL 策略：使用路径前缀 — `wangmo.tech/` (中文) / `wangmo.tech/en/` (英文)
- 语言偏好记入 localStorage，下次访问自动应用
- 首次访问：根据 `navigator.language` 自动选择
