## 左栏详细设计

### 结构（从上到下）

1. **头像区**
   - 圆形或圆角方形头像，约 120x120px
   - 可选：头像边框带微光呼吸动效（赛博感）

2. **姓名**
   - 主标题：Mo Wang（英文名，较大字号）
   - 副标题：王墨（中文名，较小字号，低对比度）

3. **身份标签**
   - "独立开发者" / "Indie Developer"
   - 可选加一句 tagline

4. **个人简介**
   - 2-3 段简短文字
   - 浅色字在深色背景上

5. **社交链接**
   - 三个图标按钮横排：即刻 / 小红书 / GitHub
   - Hover 时图标亮起（对应平台色或当前主题色）

6. **语言切换**
   - 底部，文字按钮：`中文 / EN`
   - 当前语言高亮

### 背景联动

- 左栏整体有一个半透明的渐变背景
- 渐变色 = 当前右栏聚焦作品的主题色（低饱和度版本）
- 切换作品时，背景色平滑过渡（transition 约 0.6s）
- 示例：
  - 聚焦赛博生活 → 左栏底部泛青色微光
  - 聚焦爪印档案 → 左栏底部泛暖橙微光
  - 聚焦 Lorebook → 左栏底部泛紫色微光

## 右栏详细设计

### 作品卡片

每张卡片的结构：

```
+-----------------------------------------------+
|                                                 |
|   ┌────────┐                                    |
|   │        │   作品名称                          |
|   │  图标   │   一句话描述                        |
|   │        │   [状态标签: 开发中 / Coming Soon]   |
|   └────────┘                                    |
|                                                 |
|   缩略预览图（截图或概念图，窄条横幅）             |
|                                                 |
+-----------------------------------------------+
```

- 卡片背景：深灰半透明（#1a1a2e 类似）
- 卡片边框：1px 主题色微光边框
- 状态标签样式：
  - 开发中 → 主题色填充小标签
  - Coming Soon → 虚线边框 + 低对比度文字

### 卡片交互

- **Hover**：卡片轻微上浮（translateY -4px）+ 边框亮度增强
- **聚焦判定**：右栏滚动时，视口中央的卡片为"当前聚焦"，触发左栏背景色变化
- **点击**：展开作品详情（见 03-project-detail.md）

### 滚动行为

- 右栏独立滚动，左栏固定不动
- 滚动到底部时，如果有媒体报道板块，自然衔接

## 配色系统

### 基础色

| 角色 | 色值 | 用途 |
| --- | --- | --- |
| 背景基底 | #0a0a0f | 全局最深背景 |
| 卡片背景 | #1a1a2e | 卡片、面板 |
| 主文字 | #e0e0e0 | 正文 |
| 次文字 | #888888 | 辅助信息 |
| 边框 | #2a2a3e | 默认边框 |

### 作品主题色

| 作品 | 主色 | 用于 |
| --- | --- | --- |
| 赛博生活 | #00F0FF (Cyan) | 左栏联动、卡片边框、标签 |
| 爪印档案 | #FF8C42 (暖橙) | 同上 |
| Lorebook | #8B5CF6 (紫罗兰) | 同上 |

### 左栏背景联动算法

- 取主题色，降低饱和度至 20%，降低亮度至 8%
- 作为径向渐变从底部中心扩散
- `radial-gradient(ellipse at 50% 100%, {themeColorDim} 0%, transparent 70%)`
