## 与桌面端的差异

### 头部区域（原左栏精简版）

- 头像隐藏或缩小为 48px 内联在姓名左侧
- 个人简介折叠，默认显示一行，可展开
- 社交链接图标缩小，横排
- 语言切换移到右上角

### 布局变化

- 不分栏，全部纵向排列
- 头部区域可随页面滚动（不固定）
- 也可做成轻量吸顶：滚动后头部缩为一条窄栏（姓名 + 语言切换）

### 背景联动

- 移动端简化：整个页面背景保持统一深色
- 不做随作品切换的背景联动（移动端滚动频繁，频繁变色体验差）
- 仅在作品卡片本身带一条主题色左边框作为视觉区分

### 作品卡片

- 全宽卡片，纵向排列
- 左侧带 3px 主题色竖条边框
- 点击展开详情（同桌面端，但全屏模态）

## 平板端（768px - 1023px）

- 使用移动端布局但卡片可双列
- 头部区域保留更多信息（头像 + 简介不折叠）
- 背景联动可选择性保留
