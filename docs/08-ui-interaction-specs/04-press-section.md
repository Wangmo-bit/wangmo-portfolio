# 媒体报道板块交互稿

## 显示条件

- 有报道数据时显示，无数据时整个板块隐藏（不显示空状态）

## 位置

- 桌面端：右栏底部，作品列表之后
- 移动端：作品卡片列表之后

## 布局

### 桌面端

```
+-----------------------------------------------+
|                                                 |
|   ── 媒体报道 / Press ──                       |
|                                                 |
|   ┌─────────────────────────────────────────┐  |
|   │  📰 报道标题 1                           │  |
|   │  来源：某科技媒体  ·  2026-03-15        │  |
|   └─────────────────────────────────────────┘  |
|                                                 |
|   ┌─────────────────────────────────────────┐  |
|   │  📰 报道标题 2                           │  |
|   │  来源：某公众号  ·  2026-04-01           │  |
|   └─────────────────────────────────────────┘  |
|                                                 |
|   (更多报道纵向排列，可滚动)                     |
|                                                 |
+-----------------------------------------------+
```

### 单条报道样式

```
+--------------------------------------------------+
|                                                    |
|  报道标题（一行，超长截断）             [↗ 外链]  |
|  来源名称  ·  发布日期                             |
|                                                    |
+--------------------------------------------------+
```

- 整行可点击，跳转到原始报道（新标签页打开）
- Hover 时背景微亮 + 右侧外链箭头显现
- 左侧可选带来源 favicon

## 报道数据结构

```
title_zh: 独立开发者王墨：用赛博朋克重新定义订阅管理
title_en: Indie dev Mo Wang redefines subscription management
source: 少数派
url: https://...
date: 2026-03-15
```

## 交互细节

- 报道按日期倒序排列（最新在上）
- 超过 5 条时显示"查看更多"折叠
- 移动端样式相同，全宽卡片
